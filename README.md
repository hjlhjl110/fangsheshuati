# X射线诊断刷题应用

一个基于Vue 3的移动端刷题应用，专为医用X射线诊断与介入放射学学习设计。

## 功能特性

### 📚 题库管理
- ✅ 完整提取了858道题目（含20道图片题）
- ✅ 自动清除题干中的答案标记
- ✅ 支持图片题目显示（图片正确显示在选项中）
- ✅ 题目分类和索引
- ✅ 修复了图片位置错误的问题（如第二题的图片选项）

### 🎯 刷题功能
- ✅ 清晰的题目展示界面
- ✅ 支持上一题/下一题导航
- ✅ 答题后立即反馈正确性
- ✅ 进度条显示当前进度
- ✅ 支持跳转到指定题目

### 📊 数据统计
- ✅ 实时显示答题正确率
- ✅ 记录每日学习数据
- ✅ 练习结果统计
- ✅ 学习时长记录

### 📖 错题本
- ✅ 自动收集答错题目
- ✅ 显示错误次数和最后答题时间
- ✅ 支持错题复习模式
- ✅ 按错误频率和时间筛选
- ✅ 支持错题详情查看

### 📱 移动端优化
- ✅ 响应式设计，适配各种移动设备
- ✅ 触摸手势支持（左右滑动切换题目）
- ✅ 防止误触和缩放
- ✅ 流畅的动画效果
- ✅ 优化的触摸反馈

### 💾 数据存储
- ✅ 使用localStorage保存答题记录
- ✅ 错题本数据持久化
- ✅ 学习历史记录
- ✅ 明确的数据风险提示

## 技术栈

- **前端框架**: Vue 3 + Composition API
- **路由管理**: Vue Router 4
- **状态管理**: Pinia
- **样式框架**: Tailwind CSS
- **构建工具**: Vite
- **编程语言**: TypeScript

## 项目结构

```
src/
├── assets/          # 静态资源
├── components/      # 通用组件
├── composables/     # 组合式函数
├── router/          # 路由配置
├── stores/          # 状态管理
├── types/           # TypeScript类型定义
├── utils/           # 工具函数
└── views/           # 页面组件
```

## 使用说明

1. **首页**: 显示应用概览和学习统计
2. **开始刷题**: 进入题目练习模式
3. **答题**: 选择答案后立即显示正确性反馈
4. **错题本**: 查看和管理答错的题目
5. **结果页**: 显示练习完成后的详细统计

## 数据说明

- 题库数据来自`data/questions.json`
- 图片资源存储在`public/images/xray/`目录
- 用户数据保存在浏览器localStorage中

## 注意事项

⚠️ **重要提醒**: 用户数据存储在本地浏览器中，清理浏览器缓存可能导致数据丢失。建议用户定期备份重要数据。

## 开发规范

- 遵循Vue 3 Composition API最佳实践
- 使用TypeScript确保类型安全
- 组件代码不超过300行
- 使用语义化的CSS类名
- 优化移动端触摸体验
